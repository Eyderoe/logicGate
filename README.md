# Logic Gate
逻辑门类，能够处理数值变动
## **支持类型**  
> ``单输入`` 3非\
> ``双输入`` 1或 2与 4或非 5与非 6异或 7同或\
> ``四输入`` 8与或非 9与或 10或与 11与非-与非 12或非-或非\
> ``自定义`` 其他类型逻辑门
## **操作**  
>``logicGate (int type, bool pinA, ...)`` 类型，初始值\
>``virnum('c',20,'p',10,'k')`` _means 20pF capacitor in 10K(Hz) circuit_\
>``virnum('r',5.1,'1')`` _means 5.1ohm resistor_
